Module system.keys¶

The keys module assigns names to the keycode constants that Phoenix sends in key events, and adds a few functions to make using them easier. This module uses the same names as the CraftOS keys API, so porting programs should be trivial.

system.keys.getName(id: number): (result: string | nil)¶

Returns the name for the specified keycode.

Parameters:

id (number) – The keycode to check

Returns:

result (string | nil) – The name (which is a key in keys), or nil if the code is invalid

system.keys.getCharacter(id: number): (result: string | nil)¶

Returns a printable representation of the keycode if available.

Parameters:

id (number) – The keycode to check

Returns:

result (string | nil) – The keycode’s character (in lowercase), or nil if the code doesn’t have a printable representation
