Module network

The network module implements functions for making and hosting connections with local and Internet-connected computers, as well as managing the network stack configuration.

Functions

connect (options) Creates a new connection to a remote server.
get (options) Connects to an HTTP(S) server, sends a GET request and waits for a response.
getData (url[, headers]) Connects to an HTTP(S) server, sends a GET request, waits for a response, and returns the data received after closing the connection.

connect (options)
Creates a new connection to a remote server.

Parameters:

  • options string or table The URI to connect with, or a table of options (see the connect syscall docs for more information)

    Handle A handle to the connection
get (options)
Connects to an HTTP(S) server, sends a GET request and waits for a response.

Parameters:

  • options string or table The URL to connect to, or a table of options (see the connect syscall docs for more information)

Returns:

    Handle The handle to the response data

Or

  1. nil If the connection failed
  2. string An error describing why the connection failed
