Module system.graphics

The graphics module provides functions to draw primitive geometry on a locked terminal object.

It supports both text and graphics mode terminals. The state of text terminals is preserved, so using these functions doesn't change the cursor position or colors.

drawCircle (term, x, y, width, height, color[, startAngle=0[, arcCircumference=2*math.pi]])
Draws an outlined circle (or arc) on screen.

Parameters:

  • term Terminal or GFXTerminal The terminal to draw on
  • x number The upper-left X coordinate to draw at
  • y number The upper-left Y coordinate to draw at
  • width number The width of the circle
  • height number The height of the circle
  • color number The color to draw with
  • startAngle number The angle to start from in radians (starting at the right side) (default 0)
  • arcCircumference number The amount of the arc to draw in radians (default 2*math.pi)

drawImage (term, x, y, image)
Draws an image on screen. The image may be either a valid graphics mode pixel region (using either string or table rows), or a blit table with {text, text color, background color} table rows (text mode only).

Parameters:

  • term Terminal or GFXTerminal The terminal to draw on
  • x number The X coordinate to draw at
  • y number The Y coordinate to draw at
  • image number The image to draw
