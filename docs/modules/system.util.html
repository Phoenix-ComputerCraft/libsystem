Module system.util

The util module contains various functions that don't have any specific system function, or help improve the usability of the general system.

argparse (arguments, ...)
Takes a list of valid arguments + the arguments to a program, and returns a table with the extracted arguments (and values if requested). If an argument with all `-`s is passed, processing of arguments stops, and all subsequent arguments are added to the list.

Parameters:

  • arguments {[string]=string, boolean or nil} A list of arguments that the program accepts. Single-character arguments are handled through `-a`, and longer arguments are handled through `--argument`. If the value is a truthy value, this argument requires a parameter; if the value is `"number"`, the argument requires a number parameter; if the value is `false`, the argument does not take a parameter; if the value is `nil`, the argument does not exist and will throw an error if passed.
  • ... string The arguments as passed to the program.

Returns:

    {[string]=string, number, boolean or nil,string...} The arguments as parsed from the arguments table as key-value entries, plus positional arguments as list entries.

Or

  1. nil If the arguments passed are invalid.
  2. string An error string describing what was invalid, which can be printed for the user.
timer (time)
Starts a timer that will run for the specified number of seconds. A timer event will be queued on completion.

Parameters:

  • time number The number of seconds to wait until sending the event

Returns:

    number The ID of the newly created timer
alarm (time)
Starts an alarm that will run until the specified time. A timer event will be queued on completion.

Parameters:

  • time number The time to send the event at

Returns:

    number The ID of the newly created alarm
cancel (id)
Cancels a timer or alarm. This prevents the event from triggering.

Parameters:

  • id number The ID of the timer or alarm to cancel
sleep (time)
Pauses the process for a certain amount of time.

Parameters:

  • time number The amount of time to wait for, in seconds
filterEvent (...)
Waits until an event of the specified type(s) occurs.

Parameters:

  • ... string The event names to filter for

Returns:

  1. string The event type that was matched
  2. table The parameters for the event
queueEvent (event, param)
Queues an event to loop back to the process.

Parameters:

  • event string The event name to send
  • param table The parameter table to send with the event
split (str[, sep="%s"])
Splits a string into components.

Parameters:

  • str string The string to split
  • sep string The delimiter match class to split by (default "%s")

Returns:

    {string...} The components of the string
copy (value)
Copies a value recursively, including all its keys and values.

Parameters:

  • value any The value to copy

Returns:

    any A copy of the value, with all keys, values, and metatables duplicated.
addEventListener (event, callback)
Adds an event listener to the listening module.

Parameters:

  • event string The event to listen for
  • callback function(string,table):boolean The function to call when the event is queued. If the function returns a truthy value, processing for the current event will stop. If the function throws an error, the loop will stop.
removeEventListener (event, callback)
Removes an event listener from the listening module.

Parameters:

  • event string The event to listen for
  • callback function(string,table) The function to remove
runEvents ()
Runs the event listening loop on the current thread, blocking forever.

Returns:

    string The error that caused the function to stop
startEvents ()
Runs the event listening loop on a new thread, allowing code to run after.

Returns:

    number The ID of the new thread
type (value)
Returns the type of the parameter, with the ability to check the __name metamethod for custom types.

Parameters:

Returns:

    string The type of the value
crc32 (str[, polynomial=0xEDB88320[, crc=0xFFFFFFFF]])
Calculates the CRC-32 checksum of the specified data.

Parameters:

  • str string The data to checksum
  • polynomial table or number The polynomial for the CRC, or the lookup table to use (default 0xEDB88320)
  • crc number The initial CRC value (default 0xFFFFFFFF)

Returns:

    number The calculated CRC checksum
