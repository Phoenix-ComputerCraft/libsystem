Module system.ipc

The IPC module provides functions for sending messages to other processes.

Sends an event to the owner of a service.

  • event string The event name to send
  • param table The parameter table to send with the event

receiveEvent ([pid[, event[, timeout]]])
Waits for a remote event, filtering for processes or event names, with an optional timeout.

Parameters:

  • pid number The PID to wait for an event from (optional)
  • event string The event to filter for (optional)
  • timeout number The maximum number of seconds to wait for (optional)

Returns:

  1. string The event name received
  2. table The parameters for the event

Or

    nil If the function timed out
