Modules

expect The expect module provides error checking functions for other libraries.
filesystem The filesystem module implements common operations for working with the filesystem, including wrappers for syscalls.
graphics The graphics module provides functions to draw primitive geometry on a locked terminal object.
hardware The hardware module implements functions for operating on peripherals and other hardware devices.
keys The keys module assigns names to the keycode constants that Phoenix sends in key events, and adds a few functions to make using them easier.
log The log module exposes functions for interacting with the logging subsystem.
network The network module implements functions for making and hosting connections with local and Internet-connected computers, as well as managing the network stack configuration.
pretty Provides a "pretty printer", for rendering data structures in an aesthetically pleasing manner.
process The process module allows querying various properties about the current process, as well as creating, modifying, and searching other processes.
terminal The terminal module defines functions to allow interacting with the terminal and screen, as well as handling user input.
util The util module contains various functions that don't have any specific system function, or help improve the usability of the general system.
